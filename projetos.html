DevCycle - Front-end

  • DevCycle é um site responsivo criado para promover os serviços de uma agência de desenvolvimento web, com foco em oferecer uma experiência visual moderna e impactante. O layout é totalmente adaptável a diferentes dispositivos e utiliza animações suaves para tornar a navegação mais fluida e envolvente, valorizando a apresentação dos conteúdos e o profissionalismo da marca.

  • Tecnologias utilizadas: HTML5, CSS3, Sass, JavaScript, GSAP.

Escola - Back-end

  • Sistema Escolar é uma aplicação web desenvolvida com o objetivo de simular o gerenciamento de uma escola, integrando tecnologias modernas tanto no front-end quanto no back-end. O projeto foi construído com um framework próprio em PHP, seguindo a arquitetura MVC (Model-View-Controller), e permite o cadastro, controle e visualização de informações escolares de forma prática e organizada.

  • Tecnologias utilizadas: HTML5, CSS3, Sass, JavaScript, Bootstrap, Swiper.js, Smooth Scrollbar, PHP (com framework próprio), MySQL.
